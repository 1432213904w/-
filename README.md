## 如何部署

+ 第一步：按照提供的集合文件依次创建集合
+ 第二步：集合创建好之后，依次将json文件导入对应集合
+ 第三步：依次修改集合权限为所有用户可读，仅创建者可读写；
+ 第四步：以下四个集合notes、favor、profiles、memory，设置为自定义安全规则，并且都改为true
+ 第五步：依次上传云函数

第三步、第四步看下面权限示例里面的截图，按照截图配置
